# リリースノート v1.2 (2026/02/05)

## 変更点

### UI/UXの改善
- ツリーの「独立（昇格）」操作を行った際、ツリー一覧リストが即座に更新され、作成された新しいツリーへ自動的に画面が切り替わるように改善しました。
- モバイル版において、アプリケーションの初期ロード中に「新規作成ボタン」が一瞬表示されてしまう問題を修正し、ローディング表示が適切に行われるようにしました。

### バグ修正
- 要素を編集・更新した後に、詳細メニュー（リサーチや削除など）が表示されなくなる問題を修正しました。
- 編集画面の入力欄に、テキストではなくシステム内部のオブジェクト形式が表示されてしまう不具合を修正しました。
